# Smart Expense Tracker Web Application

## Overview

Smart Expense Tracker is a web-based personal finance management application that helps users track expenses, monitor budgets, and visualize spending patterns through interactive charts and dashboards.

The application provides a simple and user-friendly interface for managing daily transactions and analyzing financial data.

---

## Features

* User Login Page
* User Registration Page
* Dashboard Overview
* Add New Transactions
* Edit Existing Transactions
* Delete Transactions
* Budget Monitoring
* Income & Expense Summary
* Expense Category Pie Chart
* Income vs Expense Bar Chart
* Local Storage Data Persistence
* Responsive User Interface
